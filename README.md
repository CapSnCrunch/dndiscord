# DnDiscord

A web application for managing Discord-based Dungeons & Dragons campaigns.

## Overview

DnDiscord bridges your World Anvil campaigns to Discord. Sync your campaign data including NPCs, maps, items, and lore directly into your Discord server with AI-powered interactions and automated game management.

## Features

- **World Anvil Sync** - Automatically sync your campaign data from World Anvil
- **Discord Bot Integration** - Bring your campaign content directly into Discord
- **AI-Powered NPCs** - Generate dynamic NPC dialogue and interactions
- **Automated Game Management** - Streamline session preparation and gameplay
- **Real-time Updates** - Keep your Discord server in sync with World Anvil changes

## Tech Stack

- **React + TypeScript** - Modern web application framework
- **discord.js** - Discord bot integration
- **World Anvil API** - Campaign data synchronization
- **OpenAI API** - AI-generated content and interactions
- **Firebase** - Authentication, database, and hosting

## Getting Started

### Client Application

```bash
cd client
npm install
npm run dev
```

The client will be available at `http://localhost:5173`
